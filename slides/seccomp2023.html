Expressões

Uma expressão é geralmente formada como uma composição de outras expressões

De acordo com as regras de precedência, temos a seguinte ordem de execução realizada pelo Python:

  1. A expressão 10 - 2 é resolvida;
  2. O resultado é utilizando para resolver a expressão de multiplicação 4 * ...;
  3. Enfim, o resultado é utilizado para resolver a expressão de soma 1 + ..., que é a expressão "original"

Os tipos int e float

Tipos Numéricos

int: números inteiros

Os inteiros podem ser representados utilizando diferentes bases:

Decimal (base 10): 1 -1 42 16 -16 Hexadecimal (base 16): 0x1 -0x1 0x2A 0x10 -0x10 Octal (base 8): 0o1 -0o1 0o52 0o20 -0o20 Binários (base 2): 0b1 -0b1 0b101010 0b10000 -0b10000

Tipos Numéricos

float: números de ponto flutuante

Números que possuem uma parte fracionária.

Alguns exemplos de representações:

3.14
Utilizando partes inteira e facionária
10.
Omitindo parte fracionária, equivalente a 10.0
.001
Omitindo a parte inteira, equivalente a 0.001
3.14e-10
Forma exponencial, representando o valor 3,14 ⋅ 10 -10

Tipos Numéricos

float: números de ponto flutuante

Note que a representação precisa pelo menos ter o ponto decimal (i.e. .) ou a parte exponencial (i.e. e), senão o valor será considerado um int:

789.0 é um float

789e0 é um float

789 é um int

Há diversos tipos de erros:

  • ValueError: Ocorre quando um valor utilizado em uma operação é inválido. Esse tipo de exceção é lançada pela função int(), por exemplo.

Dicionários

Dicionários

Um dicionário faz um mapeamento de um conjunto de chaves para respectivos valores.

Cada item do dicionário é um par chave→valor.

Qualquer tipo imutável (e.g. str int tuple) pode ser utilizado como chave, e qualquer outro tipo pode ser usado como valor.

Dicionários

Um uso comum de dicionários é para representação de alguma entidade com diferentes atributos:

Coleções

Todos esses tipos de coleções são iteráveis, isso significa que é possível percorrer seus valores utilizando for, por exemplo.

Síntese de Coleções

Para todas as coleções é possível utilizar a função len() para descobrir a quantidade de elementos contidos nas mesmas.

Para as Listas e Tuplas, o acesso de um elemento ou uma fatia de seus elementos é feito utilizando número (ou slice) entre colchetes (e.g. lista[1], tupla[:3])

Síntese de Coleções

Tuplas: Recomendado quando a ordem dos valores tem significado; permite acesso individual de itens ou por slices; imutáveis

Listas: Recomendado quando se pretende alterar a ordem, acrescentar ou remover itens; permite acesso individual de itens ou por slices; mutáveis

Síntese de Coleções

Dicionários: Recomendado quando se deseja indexar pares de chave-valor; permite acesso individual de itens; mutáveis

Conjuntos: Recomendado quando se deseja registrar ocorrência de elementos sem manter repetições; mutável

Tanto conjuntos quanto dicionários possuem rápido tempo de busca por chave. Para esse propósito são preferíveis em relação a tuplas e listas.
